# 项目说明

## 当前已确定
- 书名：《重回20岁，我改变了嫂子命运》
- 核心关系：男主重生后守护哥哥未婚妻，不写已婚嫂子关系
- 主角组：林川 / 江予柔 / 林河
- 哥哥事故线：走“被人坑导致事故”方向
- 主打法：赚钱爽点 + 命运倒计时 + 压抑守护感

## 前30章核心推进
- 前3章必须同时完成：重生、倒计时、第一桶金苗头
- 第一桶金固定为：活动赠品与电子小配件短期囤货倒卖
- 第一阶段目标不是直接救成，而是让林川先意识到：
  - 哥哥不是单纯倒霉
  - 是被一步步推到事故口上的
- 情感线前30章只做：
  - 信任建立
  - 依赖萌芽
  - 克制拉扯
  - 不进入越界表达

## 最新剧情推进摘要
- 第1章推进：林川从哥哥林河死亡、江予柔人生崩塌、自己活废掉的前世记忆中惊醒，确认自己重生回2011年7月18日，距离林河出事还剩37天；章尾张勇来电落下第一记硬钩子。
- 第2章推进：江予柔以“活着、完整、还站在晨光里”的状态首次登场，林川强压住前世情绪，确认单靠劝阻没人会信，必须先有钱和筹码；海西广场促销节成为第一桶金起手点。
- 第3章推进：林川去城郊市场扫第一批便宜充电头和小风扇，提前在工地外围撞见“刹车片没换利索”“后天换人顶车”的对话，确认事故链不是临时天灾，而是换车、换班、换人的连续做局。
- 第4章推进：林川开始背货出货、押物凑本、继续补雨披，第一次在现实里展现执行力；林河也第一次认真听见弟弟提到“那车有问题”。
- 第5章推进：暴雨夜卖货成功，林川赚到重生后的第一笔实钱并把钱拍到家里桌上，暂时撬动了家中话语权；同时，马成彪第一次以更深层包工头身份进入事故链。

## 1-5章阶段判断
- 已完成“重生回归 + 倒计时建立 + 江予柔首次登场 + 第一桶金起手 + 事故链第一层露头”这一小阶段。
- 当前读者奖励已给到：
  - 重生冲击
  - 江予柔活着出现的情绪拉力
  - 第一桶金真正落袋
  - 哥哥事故不是单纯意外的第一层征兆
- 当前情感边界仍然稳定：林川对江予柔是强烈守护与前世亏欠感，尚未出现越界表达。

## 结构重点
- 每5章都要给读者奖励：
  - 钱到账
  - 预判成功
  - 事故线推进
  - 情感压抑钩子
- 第一阶段结尾必须让读者知道：
  - 真正的事故线远比男主记忆里复杂
  - 男主改掉一层，还会露出下一层
