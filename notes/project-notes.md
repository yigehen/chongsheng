# 项目说明

## 当前已确定
- 书名：《重回20岁，我改变了嫂子命运》
- 核心关系：男主重生后守护哥哥未婚妻，不写已婚嫂子关系
- 主角组：林川 / 江予柔 / 林河
- 哥哥事故线：走“被人坑导致事故”方向
- 主打法：赚钱爽点 + 命运倒计时 + 压抑守护感

## 前30章核心推进
- 前3章必须同时完成：重生、倒计时、第一桶金苗头
- 第一桶金固定为：活动赠品与电子小配件短期囤货倒卖
- 第一阶段目标不是直接救成，而是让林川先意识到：
  - 哥哥不是单纯倒霉
  - 是被一步步推到事故口上的
- 情感线前30章只做：
  - 信任建立
  - 依赖萌芽
  - 克制拉扯
  - 不进入越界表达

## 最新剧情推进摘要
- 第1章推进：林川从哥哥林河死亡、江予柔人生崩塌、自己活废掉的前世记忆中惊醒，确认自己重生回2011年7月18日，距离林河出事还剩37天；章尾张勇来电落下第一记硬钩子。
- 第2章推进：江予柔以“活着、完整、还站在晨光里”的状态首次登场，林川强压住前世情绪，确认单靠劝阻没人会信，必须先有钱和筹码；海西广场促销节成为第一桶金起手点。
- 第3章推进：林川去城郊市场扫第一批便宜充电头和小风扇，提前在工地外围撞见“刹车片没换利索”“后天换人顶车”的对话，确认事故链不是临时天灾，而是换车、换班、换人的连续做局。
- 第4章推进：林川开始背货出货、押物凑本、继续补雨披，第一次在现实里展现执行力；林河也第一次认真听见弟弟提到“那车有问题”。
- 第5章推进：暴雨夜卖货成功，林川赚到重生后的第一笔实钱并把钱拍到家里桌上，暂时撬动了家中话语权；同时，马成彪第一次以更深层包工头身份进入事故链。
- 第6章推进：家里第一次因为林川赚到的钱真正喘了口气，母亲开始敢先留药钱和买肉补两天；但张勇没有放弃林河，而是继续用“先不定、随时顶上”的方式把他挂在危险边上。
- 第7章推进：林川借帮江予柔搬米、跑腿的机会，第一次更自然地进入她的现实生活；江予柔也开始把“盯着林河不对劲”这件事正式托给林川，双方建立起第一层实在的信任。
- 第8章推进：工地小三轮掉扣件，林川提前一嗓子让林河躲开，第一次真正改掉一场眼前的小事故；张勇也因此开始注意林川，说明林川的动作已经进入对方视线。
- 第9章推进：林川绕到废料场听见工人议论，确认老贺在混旧件、压价材料，事故被进一步坐实为“旧材料 + 坏车 + 乱调班”构成的一整条被压缩的安全链；章尾已经推进到“有人准备让林河顶一天班”。
- 第10章推进：林川开始用清尾货、转卖小台灯和耳机等方式，把第一桶金往更稳的小买卖上滚；江予柔第一次真心夸他“是真的长大了”，可这句夸反而让林川更难受；与此同时，张勇那边也正式把“后天下午顶一天班”往林河身上落。

## 1-10章阶段判断
- 已完成“重生回归 + 倒计时建立 + 江予柔入场 + 第一桶金起手并滚动 + 工地小事故先躲开一次 + 顶班危险真正落到眼前”这一开篇两小段。
- 当前读者奖励已给到：
  - 重生冲击
  - 江予柔活着出现的情绪拉力
  - 第一桶金真正落袋并开始继续滚动
  - 林川第一次用预判躲开现实里的一次小事故
  - 哥哥事故不是单纯意外，而是一整条被压缩安全链的系统性做局
- 当前情感边界仍然稳定：林川对江予柔的情绪依然落在守护、亏欠、克制与心疼上，没有越界表达；江予柔对林川的认知则从“冲动弟弟”开始慢慢转向“能帮忙、能依赖一点的人”。

## 结构重点
- 每5章都要给读者奖励：
  - 钱到账
  - 预判成功
  - 事故线推进
  - 情感压抑钩子
- 第一阶段结尾必须让读者知道：
  - 真正的事故线远比男主记忆里复杂
  - 男主改掉一层，还会露出下一层
