# 项目说明

## 当前已确定
- 书名：《重回20岁，我改变了嫂子命运》
- 核心关系：男主重生后守护哥哥未婚妻，不写已婚嫂子关系
- 主角组：林川 / 江予柔 / 林河
- 哥哥事故线：走“被人坑导致事故”方向
- 主打法：赚钱爽点 + 命运倒计时 + 压抑守护感

## 前30章核心推进
- 前3章必须同时完成：重生、倒计时、第一桶金苗头
- 第一桶金固定为：活动赠品与电子小配件短期囤货倒卖
- 第一阶段目标不是直接救成，而是让林川先意识到：
  - 哥哥不是单纯倒霉
  - 是被一步步推到事故口上的
- 情感线前30章只做：
  - 信任建立
  - 依赖萌芽
  - 克制拉扯
  - 不进入越界表达

## 最新剧情推进摘要
- 第1章推进：林川从哥哥林河死亡、江予柔人生崩塌、自己活废掉的前世记忆中惊醒，确认自己重生回2011年7月18日，距离林河出事还剩37天；章尾张勇来电落下第一记硬钩子。
- 第2章推进：江予柔以“活着、完整、还站在晨光里”的状态首次登场，林川强压住前世情绪，确认单靠劝阻没人会信，必须先有钱和筹码；海西广场促销节成为第一桶金起手点。
- 第3章推进：林川去城郊市场扫第一批便宜充电头和小风扇，提前在工地外围撞见“刹车片没换利索”“后天换人顶车”的对话，确认事故链不是临时天灾，而是换车、换班、换人的连续做局。
- 第4章推进：林川开始背货出货、押物凑本、继续补雨披，第一次在现实里展现执行力；林河也第一次认真听见弟弟提到“那车有问题”。
- 第5章推进：暴雨夜卖货成功，林川赚到重生后的第一笔实钱并把钱拍到家里桌上，暂时撬动了家中话语权；同时，马成彪第一次以更深层包工头身份进入事故链。
- 第6章推进：家里第一次因为林川赚到的钱真正喘了口气，母亲开始敢先留药钱和买肉补两天；但张勇没有放弃林河，而是继续用“先不定、随时顶上”的方式把他挂在危险边上。
- 第7章推进：林川借帮江予柔搬米、跑腿的机会，第一次更自然地进入她的现实生活；江予柔也开始把“盯着林河不对劲”这件事正式托给林川，双方建立起第一层实在的信任。
- 第8章推进：工地小三轮掉扣件，林川提前一嗓子让林河躲开，第一次真正改掉一场眼前的小事故；张勇也因此开始注意林川，说明林川的动作已经进入对方视线。
- 第9章推进：林川绕到废料场听见工人议论，确认老贺在混旧件、压价材料，事故被进一步坐实为“旧材料 + 坏车 + 乱调班”构成的一整条被压缩的安全链；章尾已经推进到“有人准备让林河顶一天班”。
- 第10章推进：林川开始用清尾货、转卖小台灯和耳机等方式，把第一桶金往更稳的小买卖上滚；江予柔第一次真心夸他“是真的长大了”，可这句夸反而让林川更难受；与此同时，张勇那边也正式把“后天下午顶一天班”往林河身上落。
- 第11章推进：兄弟第一次真正正面硬撞。林川把“你要眼看着我去死吗”这种近乎失控的话砸出来，硬把林河从立刻点头拖成“先给你一天半”。
- 第12章推进：林川顺着工地外围和小卖部线摸到司机冯二，确认蓝皮车、责任切割和顶班安排是一套提前想过的逻辑；更关键的是，冯二这条司机线与林川前世“事故后失踪的人影”真正对上了。
- 第13章推进：赚钱线再次给读者奖励。林川把耳机、小配件和旧货市场的门路继续滚起来，不只给家里再留一笔钱，还买回一台电风扇，第一次把“能挣钱”转成“能补家”；林河也第一次被逼到说出“我再想想”。
- 第14章推进：江予柔终于直接问出“你是不是早就知道”，说明她已经察觉林川不是普通地担心；林川没法说破，只能半遮半掩撑住，而两人之间“信一点、也更担心一点”的张力真正成形。章尾则推进到张勇把时间再次提前。
- 第15章推进：林川第一次正面撞进张勇吃饭的场子，把蓝皮车、冯二、材料、顶班和责任切割线当面挑破；张勇最终翻脸，把那句最冷的话扔出来——“你哥这人老实，出了事也最方便背。”至此，林河终于亲耳听见自己在那群人眼里到底是什么位置。
- 第16章推进：林川意识到自己之前还是低估了局势，真正问题不只是某次事故，而是林河从头到尾都被挑成了最适合顶上去的那个人；林河也第一次明确说出“明天不去那趟”，但新的活路仍然没找到。
- 第17章推进：江予柔家庭压力线展开。她把订婚相关的账单、礼数和现实压力摊开给林川看，也第一次让林川真正看清，她不是只会站在旁边等的人，她也一直在扛，只是能动的空间太小。
- 第18章推进：张勇那边的人直接堵到林家门口施压，想把林河重新逼回去；林家这次当街把这波逼门顶了回去，林河也更清楚自己确实是被人往坑里送；但章尾同时落下更硬的钩子——真正大的那趟活，时间提前了。
- 第19章推进：林川为了给家里和林河争出更多缓冲，再次压上一波活动货，短期暴赚一小笔；爽点落地了，可他很快意识到，钱虽然多了，离“彻底改命”还是不够。
- 第20章推进：林河找新活接连碰壁，家里再次被现实压住；这一次，江予柔第一次明确站到林川这边，直接劝林河先别接那条线的活。三个人第一次真正像一家人一样把他往回拽，但章末又落下一条陌生短信，说明命运的反扑还没停。
- 第21章推进：林川替林河摸到大学城仓储站的试活口子，第一次真正找出一条能替代工地的现实活路；但章尾张勇又找上门，说明回钩不会停。
- 第22章推进：张勇带着更高的价码上门回钩，想用“长期活 + 高收入”重新把林河拖回去；林川当面点破，真正的问题不是缺人，而是对方非要林河去，林河也第一次自己把这层意思说出来。
- 第23章推进：旧债/替班内幕正式露头。原本该去的人是赵志强，他先被马成彪那边的旧债压住，临头又因为怕死不敢上，最后借着“借班”这层人情皮，把林河一步步推成了最好用的替班对象。
- 第24章推进：江予柔主动把林川叫出去，第一次明确说出“你别把自己逼疯了”“这回我不会只站在你身后”；情感拉扯真正成形，但边界还在，压抑感也第一次真正立住。
- 第25章推进：林川盯班次，亲眼看见林河名字从调班表上被划掉，先让读者和角色同时吃到一次“命运好像真被改掉了”的短暂爽感；可章尾立刻反转——真正要命的不是原来那班，而是换过后的补班，对方要的也始终不是哪一天，而是林河这个人。
- 第26章推进：林川终于明确写出“自己只是改掉了表面时间，没改掉事故逻辑”，并把整个局拆成“人、车、活、流程”四层，真正开始从救火转成布局。
- 第27章推进：赚钱线形成第一阶段小闭环。林川把前面散出去的配件、活动货、仓储站补货和小单整合起来，第一次攒出一笔真正能让家里站稳的钱；但章尾又落下有人往门把上挂旧刹车片的警告。
- 第28章推进：替代活路突然被卡，江予柔也第一次把“你别把自己逼成这样”说得更近、更重；她不仅看见林川在扛，也开始真怕他把自己逼垮。
- 第29章推进：林川冒险混进后场办公室，摸到第一块真正的硬证据——临时补班调配单；这张蓝色复写纸第一次把“人、车、活、流程”四层逻辑钉在同一张纸上。
- 第30章推进：拿到第一张真纸后，林川顺着四层逻辑继续往上抬，终于意识到事故当天现场很可能还有一个“不该出现的人”；第一阶段就此真正收束，钩子从“怎么改班”抬到“那天到底还有谁在场”。

## 1-30章阶段判断
- 已完成“重生回归 + 倒计时建立 + 第一桶金起手并滚动 + 事故线由单点升级为系统性做局 + 替代活路初露 + 旧债/替班内幕露头 + 第一块硬证据落地 + 更上层在场者钩子浮出”这一整段前30章第一阶段推进。
- 当前读者奖励已给到：
  - 重生冲击
  - 多轮赚钱爽点与阶段性闭环
  - 小事故被提前躲开
  - 包工头亲口挑明背锅逻辑
  - 江予柔第一次明确站队林川
  - 第25章“先爽后翻”反转
  - 第29章拿到第一张真纸
  - 第30章抬出更大层级的人物钩子
- 当前情感线状态：边界仍守着，但已经从“信一点、担心一点”推进到“主动分担 + 明确心疼 + 不只站在身后”。压抑感和依赖感都已成形，后续只要再往前推半步就会更危险，也更抓人。

## 结构重点
- 每5章都要给读者奖励：
  - 钱到账
  - 预判成功
  - 事故线推进
  - 情感压抑钩子
- 第一阶段结尾已经完成：
  - 真正的事故线远比男主记忆里复杂
  - 男主改掉一层，后面立刻露出下一层
