# 项目说明

## 当前已确定
- 书名：《重回20岁，我改变了嫂子命运》
- 核心关系：男主重生后守护哥哥未婚妻，不写已婚嫂子关系
- 主角组：林川 / 江予柔 / 林河
- 哥哥事故线：走被人坑导致事故方向
- 主打法：赚钱爽点 + 命运倒计时 + 压抑守护感

## 前35章核心推进
- 前3章必须同时完成：重生、倒计时、第一桶金苗头
- 第一桶金固定为：活动赠品与电子小配件短期囤货倒卖
- 第一阶段目标不是直接救成，而是让林川先意识到：
  - 哥哥不是单纯倒霉
  - 是被一步步推到事故口上的
- 情感线前35章只做：
  - 信任建立
  - 依赖加深
  - 克制拉扯
  - 不进入越界表达
- 第31到35章必须把第一张真纸从知道真相，推进到能真正撬动局势的筹码，同时把更上层在场者从抽象猜测推进成可追查的线索链

## 最新剧情推进摘要
- 第1章推进：林川从哥哥林河死亡、江予柔人生崩塌、自己活废掉的前世记忆中惊醒，确认自己重生回2011年7月18日，距离林河出事还剩37天；章尾张勇来电落下第一记硬钩子。
- 第2章推进：江予柔以活着、完整、还站在晨光里的状态首次登场，林川强压住前世情绪，确认单靠劝阻没人会信，必须先有钱和筹码；海西广场促销节成为第一桶金起手点。
- 第3章推进：林川去城郊市场扫第一批便宜充电头和小风扇，提前在工地外围撞见刹车片没换利索和后天换人顶车的对话，确认事故链不是临时天灾，而是换车、换班、换人的连续做局。
- 第4章推进：林川开始背货出货、押物凑本、继续补雨披，第一次在现实里展现执行力；林河也第一次认真听见弟弟提到那车有问题。
- 第5章推进：暴雨夜卖货成功，林川赚到重生后的第一笔实钱并把钱拍到家里桌上，暂时撬动了家中话语权；同时，马成彪第一次以更深层包工头身份进入事故链。
- 第6章推进：家里第一次因为林川赚到的钱真正喘了口气，母亲开始敢先留药钱和买肉补两天；但张勇没有放弃林河，而是继续用先不定、随时顶上的方式把他挂在危险边上。
- 第7章推进：林川借帮江予柔搬米、跑腿的机会，第一次更自然地进入她的现实生活；江予柔也开始把盯着林河不对劲这件事正式托给林川，双方建立起第一层实在的信任。
- 第8章推进：工地小三轮掉扣件，林川提前一嗓子让林河躲开，第一次真正改掉一场眼前的小事故；张勇也因此开始注意林川，说明林川的动作已经进入对方视线。
- 第9章推进：林川绕到废料场听见工人议论，确认老贺在混旧件、压价材料，事故被进一步坐实为旧材料、坏车、乱调班构成的一整条被压缩的安全链；章尾已经推进到有人准备让林河顶一天班。
- 第10章推进：林川开始用清尾货、转卖小台灯和耳机等方式，把第一桶金往更稳的小买卖上滚；江予柔第一次真心夸他是真的长大了，可这句夸反而让林川更难受；与此同时，张勇那边也正式把后天下午顶一天班往林河身上落。
- 第11章推进：兄弟第一次真正正面硬撞。林川把你要眼看着我去死吗这种近乎失控的话砸出来，硬把林河从立刻点头拖成先给你一天半。
- 第12章推进：林川顺着工地外围和小卖部线摸到司机冯二，确认蓝皮车、责任切割和顶班安排是一套提前想过的逻辑；更关键的是，冯二这条司机线与林川前世事故后失踪的人影真正对上了。
- 第13章推进：赚钱线再次给读者奖励。林川把耳机、小配件和旧货市场的门路继续滚起来，不只给家里再留一笔钱，还买回一台电风扇，第一次把能挣钱转成能补家；林河也第一次被逼到说出我再想想。
- 第14章推进：江予柔终于直接问出你是不是早就知道，说明她已经察觉林川不是普通地担心；林川没法说破，只能半遮半掩撑住，而两人之间信一点、也更担心一点的张力真正成形。章尾则推进到张勇把时间再次提前。
- 第15章推进：林川第一次正面撞进张勇吃饭的场子，把蓝皮车、冯二、材料、顶班和责任切割线当面挑破；张勇最终翻脸，把那句最冷的话扔出来，林河终于亲耳听见自己在那群人眼里到底是什么位置。
- 第16章推进：林川意识到自己之前还是低估了局势，真正问题不只是某次事故，而是林河从头到尾都被挑成了最适合顶上去的那个人；林河也第一次明确说出明天不去那趟，但新的活路仍然没找到。
- 第17章推进：江予柔家庭压力线展开。她把订婚相关的账单、礼数和现实压力摊开给林川看，也第一次让林川真正看清，她不是只会站在旁边等的人，她也一直在扛，只是能动的空间太小。
- 第18章推进：张勇那边的人直接堵到林家门口施压，想把林河重新逼回去；林家这次当街把这波逼门顶了回去，林河也更清楚自己确实是被人往坑里送；但章尾同时落下更硬的钩子，真正大的那趟活，时间提前了。
- 第19章推进：林川为了给家里和林河争出更多缓冲，再次压上一波活动货，短期暴赚一小笔；爽点落地了，可他很快意识到，钱虽然多了，离彻底改命还是不够。
- 第20章推进：林河找新活接连碰壁，家里再次被现实压住；这一次，江予柔第一次明确站到林川这边，直接劝林河先别接那条线的活。三个人第一次真正像一家人一样把他往回拽，但章末又落下一条陌生短信，说明命运的反扑还没停。
- 第21章推进：林川替林河摸到大学城仓储站的试活口子，第一次真正找出一条能替代工地的现实活路；但章尾张勇又找上门，说明回钩不会停。
- 第22章推进：张勇带着更高的价码上门回钩，想用长期活和高收入重新把林河拖回去；林川当面点破，真正的问题不是缺人，而是对方非要林河去，林河也第一次自己把这层意思说出来。
- 第23章推进：旧债、替班内幕正式露头。原本该去的人是赵志强，他先被马成彪那边的旧债压住，临头又因为怕死不敢上，最后借着借班这层人情皮，把林河一步步推成了最好用的替班对象。
- 第24章推进：江予柔主动把林川叫出去，第一次明确说出你别把自己逼疯了和这回我不会只站在你身后；情感拉扯真正成形，但边界还在，压抑感也第一次真正立住。
- 第25章推进：林川盯班次，亲眼看见林河名字从调班表上被划掉，先让读者和角色同时吃到一次命运好像真被改掉了的短暂爽感；可章尾立刻反转，真正要命的不是原来那班，而是换过后的补班，对方要的也始终不是哪一天，而是林河这个人。
- 第26章推进：林川终于明确写出自己只是改掉了表面时间，没改掉事故逻辑，并把整个局拆成人、车、活、流程四层，真正开始从救火转成布局。
- 第27章推进：赚钱线形成第一阶段小闭环。林川把前面散出去的配件、活动货、仓储站补货和小单整合起来，第一次攒出一笔真正能让家里站稳的钱；但章尾又落下有人往门把上挂旧刹车片的警告。
- 第28章推进：替代活路突然被卡，江予柔也第一次把你别把自己逼成这样说得更近、更重；她不仅看见林川在扛，也开始真怕他把自己逼垮。
- 第29章推进：林川冒险混进后场办公室，摸到第一块真正的硬证据，临时补班调配单；这张蓝色复写纸第一次把人、车、活、流程四层逻辑钉在同一张纸上。
- 第30章推进：拿到第一张真纸后，林川顺着四层逻辑继续往上抬，终于意识到事故当天现场很可能还有一个不该出现的人；第一阶段就此真正收束，钩子从怎么改班抬到那天到底还有谁在场。
- 第31章推进：林川在江予柔提醒下，把查法从回忆脸改成先找交点，顺着蓝2、材料批次和到场资格往上摸；小卖部口风坐实了那个干净男人确实提前在工地附近露过头。
- 第32章推进：蓝纸第一次真正从知道真相转成撬动局势的筹码。林川逼黄顺松口，抠出周助这条新线，确认对方曾两次核对车辆和材料。
- 第33章推进：林川把真纸转移到最不显眼的地方，同时继续用赚钱线给林河补足替代活路的底气；当晚周助直接摸到林家门口，点明他们已经闻到纸落在林川手里。
- 第34章推进：赵叔带回黑色桑塔纳线索，坐实周助背后车里还有人；林川也第一次真正意识到，对方怕的不是纸本身，而是自己顺着纸摸到那辆车里的人。
- 第35章推进：林川故意放出纸已经烧掉的风声，反逼对方试探；夜里先接到压低声音的试探电话，随后又收到西河桥下的匿名邀约，车里的人正式开始往外露面。
- 第36章推进：西河桥邀约落到实场景，林川没有被牵进仓房，而是被黄顺妹妹提前截住，坐实了桥下是收口陷阱；更关键的是，车里那个人第一次有了明确称呼，许总。
- 第37章推进：林川顺着许总这条线去汽修铺摸口风，把黑色桑塔纳和城建口方向初步连上；江予柔也再次在关键时候把林河绝不能回去这句话先说出来。
- 第38章推进：赚钱线与改命线重新合流。林川替林河拿到更稳的库房长活，真正把替代活路做实；与此同时，黄顺送来更冷的新线，医院那边当年也被压过。
- 第39章推进：医院线正式打开。林川从急诊保安嘴里摸到送医拖延和病案可被动手的口风，并在急诊楼外第一次看见像周助的白衬衫年轻人出入医院。
- 第40章推进：黑色桑塔纳在医院外与副驾白衬衫同框，车里那只转打火机的手再次出现，许总的存在从猜测推进到近实锤；同时车牌后三位317也被林川记下，医院病案楼成为下一阶段核心突破口。
- 第41章推进：第40章门外来客落到实场景。上门的人不是周助，而是替许广成跑腿试探的姓孙中年人；他故意把医院和许站长两个信息往林家门口一摔，同时掉出质安站外联登记残页与住建院停车联，住建院线由影子转成实口。
- 第42章推进：林川在后街复印店与医院门口两头核实，正式拿到许广成实名、质安站身份、周助对应关系、完整车牌宁B3176，以及司机老严这条新线；许总不再只是称呼，而是能被追查的具体人。
- 第43章推进：旧事故样本被翻出来。断腿老陈证实当年西口事故后也曾出现黑桑塔纳、白衬衫年轻人和工地后拖延送医，同时还拿出质安站协调款收条，说明许广成这套收口方法并非第一次使用。
- 第44章推进：赚钱线继续给读者奖励。学校报到潮让林川和林河再垫厚一层家底，也让林河的替代活路更稳；江予柔一边记账一边主动接医院入口，情感仍守边界但分担更重。章尾匿名纸条直接把病案楼二零七推上台面。
- 第45章推进：病案楼线第一次拿到硬信息。顾姐证实老陈旧登记曾被整页重抄，原始送诊时间约七点一十八，重抄后变成七点五十二，拖延四十分钟被落成纸证；紧接着林家遭遇翻纸搜查，说明对方也察觉查到了真东西。
- 第46章推进：许广成正式从影子变成职务明确的人。林川顺住建院司机班和赵叔口风，确认许广成是县建设工程质量安全监督站副站长，周助是其身边外联跑腿，宁B3176基本就是许广成常用车；同时住建院档案室也开始有补痕迹动作。
- 第47章推进：江予柔补到更关键的医院外口供，确认老陈当年不是走最近的北路，而是绕东桥旧路进医院；这条路不是偶然绕远，更像故意留出来的拖时间通口径区间。章尾林河包里翻出匿名纸条，直接写着八点十二前过东桥，说明这套路线可能还在沿用。
- 第48章推进：家底继续被垫厚。学校报到潮再给一轮赚钱爽点，家里药钱和日用都开始松气；与此同时，对方也开始从校门口试探林河是否会再回西口。夜里纸背压痕被铅笔蹭出，老严全名严国胜终于落地。
- 第49章推进：东桥旧路被进一步坐实。桥头卖凉茶老太太记住宁B3176常在路上停等，说明这辆车开的不是救命最短路，而是方便拖时间、等后车、打通前头关系的路；严国胜赊账记录也落成可追人的实口。章尾则把更硬的当下危险抬出来，许广成明天要去西口危改验收，而名单里已经有林河。
- 第50章推进：这一批的阶段硬钩子真正落地。林河旧工装里翻出西口危改验收临时通行证，日期、林河名字、质安站红章、许广成签发、严国胜联系人同时压在一张纸上，证明林河并非临时被盯，而是已经被提前写进即将发生的局里。

## 1到50章阶段判断
- 已完成从工地事故线、医院收尾线再到住建与质安站人脉线的连续抬升，明确许广成不是外围影子，而是能同时插手验收、送医后拖、病案重抄与协调款收口的关键节点。
- 当前新增读者奖励已给到：
  - 许广成实名与副站长身份落地
  - 宁B3176完整车牌与严国胜姓名落地
  - 老陈旧事故样本坐实同一套收口流程
  - 学校报到潮连续给到赚钱爽点，替代活路和家庭底气同步增强
  - 病案楼二零七与旧登记重抄拿到第一条纸面硬信息
  - 林河名字提前出现在危改验收通行证上，当前危险从猜测变成实物
- 当前情感线状态：江予柔已经不只是稳情绪，而是能在账、路、医院入口和风险节奏上真正替林川分力；两人默契更深、压抑更重，但仍严格守在克制边界内。
