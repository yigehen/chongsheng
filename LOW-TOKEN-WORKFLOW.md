# 《重回20岁，我改变了嫂子命运》低 Token 工作模式

## 默认读取顺序
1. `outline/outline.md`
2. `characters/characters.md`
3. `notes/project-notes.md`
4. `notes/risk-control.md`
5. `PROGRESS.md`
6. `NEXT_STEP.md`
7. `chapters/chapter-index.md`
8. 最新一章正文（必要时再读前一章）

## 写作规则
- 这本书与《火光深处等春来》完全独立，禁止混用任何设定与文件。
- 每次推进优先按有界任务执行，不无限扩写。
- 每章必须同时承担：剧情推进 + 情绪钩子 + 下一步悬念。
- 少口水解释，信息尽量放进动作、抉择、冲突和后果里。
- 赚钱爽点与情感压抑必须交替给读者奖励。

## 汇报规则
只汇报真实文件级进度：
- 新建了哪些章节
- 更新了哪些设定/笔记/进度文件
- 是否已提交 git / push

## 最小同步集合
每次写作后至少同步：
- `chapters/chapter-index.md`
- `notes/project-notes.md`
- `notes/risk-control.md`
- `README.md`
- `PROGRESS.md`
- `NEXT_STEP.md`
