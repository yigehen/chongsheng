# 第111章《老周先找的人不是许广成》

那句轮不到林河看见，让屋里几个人都沉了下来。

这话不是冲文印店老板说的。

是借老板的耳朵，往他们家里扔的一块石头。石头不大，分量却够，砸不死人，专门砸人心口。要是家里人慌了，第二步就好办了。去问，去闹，去露脸，对方什么都不用做，只要看着他们自己乱。

林川没立刻出门追人，反而先问小伍。

问路的人长什么样，鞋上有没有灰，抽什么烟，说话是本地口还是外路口，手上有没有茧，问完之后站了多久，走的时候往哪边看了一眼。

小伍让他问得一愣一愣，皱着脸回想了半天，才慢慢说出来。

年纪不大，头发剃得很短，不像周助，像个跑腿的，可说话比跑腿的硬。鞋是胶底的，边上带白条，裤腿卷着一点，像刚从车间地上蹭过灰。烟没点，只在手里夹着，像是拿来装样子。口音是本地口，可尾音有点别，像常年跟车的人。

林川点了点头，心里却没有轻。

这说明对面已经开始在外圈试水。不是许广成亲自来，也不是陈秘书这种能改纸的人来。先放一个不疼不痒的小口来敲门，想看他们会不会乱。真要是家里人一急，话多了，脚乱了，仓储站那边和质安站那边很快就能接到信。

林河把外套一披，脸色很沉。

那我更得去仓储站问换班表。

江予柔看向他，声音不高，却很稳。

你一个人去，太显眼。老周那边今天既然已经往外试口，站里多半也在等你露面。你进去一问，人家未必答你，倒先记住你。

林河嘴唇动了动，到底没反驳。他不是听劝听软，是心里也明白，现在不是赌气的时候。夜里补装那件事如果真是给他准备的，那站里每一个抬眼看他的人，都可能不是单纯看人。

林河摇头。

我不往里问，就顺路看。老周今天既然急成这样，他不会只找赵志文一个。谁先和谁串上，明天表就会动。

林川盯着哥哥看了几秒，最后还是点了头。

行，我陪你到站口。你进，我守外头。别逞强，进去以后少说话，多看门口和调度窗。谁今天不该出现却出现了，谁说话说一半又收回去，先记脸。

江予柔把门边那把旧伞递过来。

天色压得低，夜里像要落雨。你们别把车停太近。还有，站口小卖部旁边那条短路能看见饭馆后窗，真要盯人，别总站正门。

林川接伞时，指尖碰到伞骨，顿了一下，只说了句知道了。

她没有再叮嘱，也没有跟着出来。她知道自己这会儿跟去，未必是帮忙，反而更容易让人认脸。分寸到了这一步，比什么都重要。

两人到仓储站时，天已经擦黑。夜班车刚进院，门口乱，正适合看人。几辆旧货车轮胎上全是泥，装卸工把麻绳往肩上一搭，骂骂咧咧往里跑，门卫一边记车号一边扯着嗓子喊人。越是这种时候，越容易把真正有心的人藏进去。

林河进去没多久，林川就绕到小卖部旁边那条短路上，借着阴影和雨棚，把正门和对面饭馆都收进眼底。

果然，没过多久，他就在外头看见了老周。

老周不是来找许广成，也不是找陈秘书。

他缩着肩，站在仓储站斜对面小饭馆门口，手里捏着一包没拆完的烟，像在等谁。那副样子看着不起眼，可林川一眼就看出来，这不是闲等。老周今天衣服换得很快，鞋上却还是白天那层灰，说明他回去没歇，绕了一圈又出来了。

几分钟后，赵志文来了。

赵志文进门的时候脚步急，掀帘子太猛，差点把门边的竹帘挂下来。他人进去了，头还往左右看了一圈，像怕被谁看见。这样的人最说明问题。要是只谈小事，不会怕成这样。

再过一会儿，周助也来了。

周助进门最慢，先在门口停了停，像是在看有没有熟脸，然后才把手往腋下一夹，慢慢走进最里头。三个人没进包间，只坐在最里头靠墙那桌。灯不亮，外人看不清嘴，可林川能看出谁在说，谁在听。老周说得最多，脖子一伸一缩，手指总往桌上点。赵志文发急，几次抬手又压下去，像是在争什么。周助一直没怎么动，只偶尔抬眼看门，像一条不咬人的狗，却比谁都盯得紧。

这场面，比任何一句话都狠。

老周先找的人，不是许广成。

因为许广成是天上那把伞，他够不着。真有脏事，也是下头人先把路垫平了，才轮得到把伞撑出来。老周先找的是脚底下这两块板。赵志文补口，周助跑线，只要这两个人先和他站成一边，他才敢往上递话。换句话说，这事还没正式上到许广成跟前，可下头已经开始围人了。

林川盯着饭馆里那张桌子，脑子转得很快。

赵志文这个人，胆子不大，脑子却活，最怕背锅。老周今天先找他，说明补装那条线已经到了要补口的时候。周助更不用说，凡是站里临时换人、夜里加车、口头通知这种不落纸的小活，十有八九都绕得过他耳朵。两个人今天能坐在一张桌上，等于把空班、跑线、压口三步，提前拴在一起了。

林川正盯着，林河从站里出来了，脸色也不好看。

表动了。

他压低声音，声音不重，却像从牙缝里挤出来的一样。

后天有一趟夜里补装，原本是临时工顶，现在名字空着。调度窗后头那张临表挂得很高，像是故意让人看不全，可我还是看见了。前一栏写了车次，后一栏写了补装，偏偏人名那一格空着。旁边原来夹着一张蓝纸，刚才有人抽走了。

林川心里一沉。

空着，比写死还吓人。

写死了，说明他们还得担风险，得看那个人会不会突然不肯上。空着，说明这就是留给林河那一口，只等人往里填。等谁点头，等谁默许，等谁把那支笔落下去，那名字就会像原本就在上头一样。

林河也看见了饭馆那桌人，眼神瞬间冷下去。

他们在商量把谁填进去。

不是疑问，是肯定。

林川没有安慰，只把他往阴影里拉了半步。

先别露。

现在最值钱的，是他们以为我们还没看懂。要是现在过去掀桌子，只会让他们把口更收紧。你今天看见空班，已经够了。剩下的，要看他们怎么填，谁来填，填之前还会不会先做一遍试车。

林河听到试车两个字，眉头一下拧住。

你怀疑他们不等后天。

林川看着饭馆那扇油腻的窗，缓缓点头。

不是怀疑，是得当成最坏的来防。钱线和事故线走到一起的时候，人命只是账上的一格。补装是明面，试车才是暗面。真要害人，不一定等名字落死才动手。

林河没再说话，只低头点了一根烟，又在刚要吸的时候把烟掐了。周围风里带着机油味和湿土味，让人胸口发闷。他知道弟弟说得对，这一局最不能做的，就是把怒气当本事。

过了片刻，饭馆那边先动了。老周起身结账，赵志文没跟，周助反而慢了半拍。三个人不是一块走的，说明桌上还有没谈拢的地方。可越是没谈拢，越说明那条空班口是真值钱。要是不值钱，根本不用拧到这个份上。

林川把他们三个人的先后顺序记在心里，转身时又回头看了一眼调度楼亮着灯的窗子。

后天夜里那趟补装，不是普通空班。

很可能就是他们准备拿来重新写林河的那道口。

而这道口一旦写成，就不只是一个班次的事。它后面接着账，接着人情，接着赔付，接着谁替谁背锅，谁拿钱堵谁的嘴。钱线已经开始冒头，事故线也已经把钩子伸出来了。

他把伞往肩上一搭，声音低到只有林河能听见。

今晚上别回站里乱晃。先回去，把认识的人名单理一遍。谁能看调度窗，谁能碰排班条，谁嘴松，谁只认钱，明天一早我们先找最容易松口的那个。

林河看了他一眼，点了点头。

两人刚要走，饭馆里头忽然又传出椅子刮地的声音。赵志文竟然一个人折回来，站在门口发了一会儿愣，像是下不定什么决心。

林川脚步一顿，眼神缓缓冷下来。

他忽然觉得，今晚这顿饭，可能还没真正散干净。
